为别人写的一个将ajax-zoom程序的展示方式迁移到wordpress上面的插件。

== Installation ==

1. Copy ajax-zoom.zip to your Wordpress plugins directory, usually `wp-content/plugins/` and unzip the file.  It will create a `wp-content/plugins/ajax-zoom/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Then, just follow the Clear-StatPress page under options.
